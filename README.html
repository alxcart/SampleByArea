Sampling plan by area

Plugin developed based on ISO 19157 and ISO 2859 standards

Plugin features

  1. The sampling plan is drawn on the layer of interest. The geometry type of the layer can be point, line, or area.
  2. The sample size (n) is determined by the total number of cells (N) generated over the area of interest.
  3. The cell size is based on the cartographic projection of the selected layer.
Notes:
  • Versao 1.0.

For more information visit: https://github.com/alxcart/SampleByArea.
